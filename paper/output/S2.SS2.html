2.2 Virtual Keyboard

So far the most used method to input math in the web is LaTeX, e.g. Wikipedia and phpBB. Unfortunately type LaTeX code in a mobile device is hard because to access some useful characters, e.g. ^, _ and \, the user need to switch the keyboard. If mobile devices have a virtual keyboard design to math that input LaTeX it should make a little easy to edit math on the web.

One of the authors will implement such keyboard for Firefox OS as part of 2014 edition of Google Summer of Code [Bugzilla1004057].
